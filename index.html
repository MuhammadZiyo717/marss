1. Box Model Nima?

Box model — bu har bir HTML element aslida bir quticha (box) ekanligini anglatadi. U content, padding, border va margin qismlaridan iborat.

2. Misol Element

Bu yerda biz Box Model elementlarini amalda ko‘ramiz.
